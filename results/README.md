# results/

This folder stores **lightweight** extracted results that can be tracked in git.

## What belongs here

| File type | Description |
|-----------|-------------|
| `*.csv` | Force-displacement curves and other tabular outputs |
| `*.json` | Extracted scalar summaries (peak load, energy, etc.) |
| `*.png` / `*.pdf` | Figures generated by `scripts/plot_results.py` |

## What does NOT belong here

* Raw `*.odb` databases — they can exceed several GB. Store them on HPC or local disk.
* `*.h5` / `*.hdf5` — large field-output arrays. Already in `.gitignore`.

## File naming

Include the model name and cycle count (if applicable) in the file name so results
remain traceable:

```
cyclic_jump_v1_fd_curve.csv
cyclic_jump_v1_peak_forces.json
cyclic_jump_v1_hysteresis.png
```
