# models/

This folder stores Abaqus input files (`.inp`) and any supporting keyword or
parameter files used to define the finite-element models.

## What belongs here

| File type | Description |
|-----------|-------------|
| `*.inp` | Abaqus input files generated by `scripts/build_model.py` |
| `*.json` / `*.yaml` | Parameter files read by build scripts |
| `*.cae` | Abaqus/CAE model database (**optional**, large — consider `.gitignore`-ing) |

## What does NOT belong here

* `*.odb` — Output databases are large binaries. Keep them on local / HPC storage.
* `*.lck`, `*.log`, `*.msg`, `*.sta` — Abaqus run-time files. Already in `.gitignore`.

## Naming

Use descriptive, version-aware names for input files, e.g.:

```
cyclic_jump_v1.inp
cyclic_jump_v1_parameters.json
```
